# Arduino Micro
* programmable via Arduino IDE
* built-in USB communication
* 20 digital input/output pins (7 can be PWM outputs, 12 as analog inputs)
* Flash Memory - 32 KB (4kB used by bootloader)
* SRAM - 2.5 KB
* EEPROM - 1 KB
* Clock Speed - 16 MHz
